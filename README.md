# Iced node editor

A node editor for [Iced](https://github.com/iced-rs/iced) GUI library. Supports iced_native renderer.

![Example](./media/screenshot.PNG)

## Features

- [x] Nodes
- [x] Connections
- [x] Panning
- [x] Zooming
- [ ] Dragging nodes
